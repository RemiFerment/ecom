## Lancement du conteneur à partir de la nouvelle image créée

Ici, nous allons créer un nouveau conteneur et exposer des ports afin d’accéder au serveur Apache depuis notre machine.
Nous allons également monter un volume dans le conteneur.

Cela permet de synchroniser un dossier de notre machine avec le conteneur.
L’objectif est de lier un dossier contenant un fichier `index.php` que le serveur Apache pourra interpréter.

```bash
docker run -d -i --name server_lamp -p 8080:80 -v "[CHEMIN VERS VOTRE DOSSIER]:/var/www/html" lamp:latest
```

**-p => port : permet de lier le port 8080 de la machine hôte au port 80 du conteneur.**
**-v => volume : permet de lier un dossier de la machine hôte à un chemin du conteneur.**

---

## Création d’un fichier `index.php` dans le dossier relié au conteneur

Dans le dossier que vous avez relié au conteneur, créez un fichier `index.php` contenant :

```php
<?php
echo "Hello World !";
```

## Lancement de MySQL et Apache

On rentre dans l'invite de commande du conteneur:
```bash
docker exec -it lamp sh```

On rentre cette commande :
```bash
service start apache2```

Puis cette commande :
```bash
service start mysql```


---

## Test

Rendez-vous sur :
`http://localhost:8080`
Vous devriez voir **Hello World !**

Vous pouvez également accéder à :
`http://localhost:8080/phpmyadmin`

Une page de connexion s’affiche.
En utilisant **root** comme identifiant et **root** comme mot de passe, vous devriez accéder au tableau de bord phpMyAdmin.
